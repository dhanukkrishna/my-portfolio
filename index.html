AUTOMATIC HAND SANITIZER USING A DRY FOGGER CIRCUIT

Performed mathematical modeling of mechanical and thermal systems. Traditional hand sanitizer dispensers require physical contact (risking cross-contamination) and dispense excessive liquid (causing waste and skin problems). This project addresses these issues by programming a dry fogging circuit for automated hand sanitization. The setup sprays sanitizer as a contactless mist using a humidifier, using less product per use while being economical, skin-friendly, and easy to construct.
